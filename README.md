# Food Delivery Receipt Generator

## Purpose

This application generates a receipt for a food delivery order. It collects customer information, calculates the subtotal, service charge, delivery charge, and displays the final total.

## Tech Stack

- Python 3
- Modular Programming
- Functions
- Input/Output

## How to Use

1. Open the terminal.
2. Navigate to the project folder.
3. Run:

python main.py

4. Enter customer information.
5. The receipt will be displayed.
